# HF Labs Documentation

Welcome to the Markdown documentation wiki.

This docs section is rendered by `/wiki/` and loads Markdown files directly from this repository under `/docs`.

## How to publish docs as you release

1. Add or update Markdown files in `/docs`.
2. Keep `index.md` as your main entry point.
3. Use relative Markdown links (for example, `./releases/index.md`, `./releases/v1.2.0.md`).
4. Open the docs wiki at `/wiki/`.
